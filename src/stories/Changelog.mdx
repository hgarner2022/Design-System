# Changelog

All notable changes to this project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

For detailed release notes and full list of changes, please visit our [GitHub Releases](https://github.com/company/design-system/releases).

## [Unreleased]
